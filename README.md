# DSA From First Principles

Personal documentation of Data Structures and Algorithms, 
built from CLRS, MIT 6.006 lecture notes, and Algorithms 
Illuminated — written entirely in my own words after working 
through each concept from scratch.

## Structure
Each file covers one layer of the curriculum in full prose — 
definition, reasoning, Big O with explanation, implementation 
from memory, and LeetCode post-mortems.

## Curriculum Order
1. Complexity and mathematical foundations
2. Arrays and strings
3. Linked lists
4. Stacks and queues
5. Hash maps and hash sets
6. Recursion and the call stack
7. Trees and binary trees
8. Heaps and priority queues
9. Sorting algorithms
10. Graphs
11. Dynamic programming
